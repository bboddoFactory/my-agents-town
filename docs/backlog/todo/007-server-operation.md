## Problem

My-agents-town should become an always-available personal workspace, not only a laptop-local experiment.

## Desired Outcome

The system can run on the 24-hour server, be reached through Tailscale, and keep logs and artifacts on the server.

## Acceptance Criteria

- [ ] The app runs on the server.
- [ ] The browser can reach it over Tailscale.
- [ ] Worker calls run on the server.
- [ ] Logs and artifacts are written on the server.
- [ ] The user can inspect current state after a restart or reconnect.
