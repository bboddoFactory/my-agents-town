# Artifact Viewer

## State

todo

## Problem

Artifact files are useful only if the user can inspect them without digging through the filesystem.

## Desired Outcome

The web UI can show task artifacts, result artifacts, and linked run/debug details.

## Acceptance Criteria

- [ ] The UI lists recent task artifacts.
- [ ] The UI can open a task artifact.
- [ ] The UI can open a result artifact.
- [ ] The UI can show run/debug details linked to the artifact.

## Notes

Keep this as a small viewer, not a polished dashboard.
